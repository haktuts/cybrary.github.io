An Introduction to Arduino: Open Source Single-Board Microcontroller

  • Posted by Bhargav
  • Date 19 May,2014 ( 210 days ago )
  • microcontroller arduino
  • 0 Comments
  • 0 Likes

Arduino is a open-source single-board microcontroller designed to create computing platforms or interactive projects using set of various hardware and software.  The Arduino hardware (board) consists of an 8-bit Atmel AVR microcontroller or a 32-bit Atmel ARM. It also features USB port, 6 analog input pins, 14 digital I/O pins and various extension slots.

Arduino allows CPU board to be connected to interchangeable add-on modules, which are known as shields. These shields communicate with the Arduino board and makes thing happen. Arduino software and  programs are written using C or C++ programming language.

The original Arduino hardware is manufactured by the Italian company Smart Projects and costs around $30. There are many Arduino clones that one can get only for $5 or less.

Arduino is a teaching tool for do-it-yourself (DIY) electronics and programming and should not be confused with Raspberry Pi, which is a full-fledged PC in itself. Instead, Arduino is a single board microcontroller, a single component of a computer which is capable for making physical computing platforms.

Just like many other microcontrollers, Arduino is not just limited to Windows OS. Arduino comes with cross-platform support and can run on Windows, Macintosh OSX, and Linux OS.

There are many Arduino hardwares or boards such as Arduino Yùn, Arduino Leonardo, Arduino Uno, Arduino Mega, Arduino Due, Arduino Ethernet, Arduino Nano and many others. Apart from the regular versions, there are many superseded, capable versions of Arduino, such as Serial Arduino, Arduino USB, Arduino Extreme, Arduino BT (Bluetooth) and Arduino NG.

Software written using Arduino are called sketches and these are regular software written in the text editor and are saved with the file extension .ino.  Arduino software and  programs are written using C or C++ programming language.

There are many commands that can be used with Arduino and these commands are categorised under five different  menus – File, Edit, Sketch, Tools and Help. Besides these Sketches, there are many additional Libraries, which are a collection of codes. These libraries can be used to connect to a sensor, display or a module. Hundreds of libraries are available for free download on the Internet and all they need is to be installed before they can be used. You can download the official Arduino hardware over here – http://arduino.cc/en/Main/Products and the official Arduino software, drivers  and libraries from here – http://arduino.cc/en/Main/Software.

Arduino is not a new concept, there are many board-microcontroller in the market – Parallax Basic Stamp, Netmedia’s BX-24, Phidgets, MIT’s Handyboard and many more. Arduino is a very powerful programming tool and can be used to build many interactive projects, either stand-alone or coupled with software or your computer.

Arudino is based on set of hardware and software and is intended for programmers, designers and anyone who is interested in creating their own DIY electronic/computer projects or may be an interactive objects (controlled via technology). Arduino is easy to use, affordable and flexible.
